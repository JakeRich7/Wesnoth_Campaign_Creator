# Wesnoth_Campaign_Creator

### Intended for 1.18. tested on 1.18.3 (my preferred version)

### I created this over a weekend with help from AI. Simple campaign importer and creator to eliminate most of the tedious, repetitive parts of Wesnoth campaign creation.

### I may continue to expand this, but we shall see. Feel free to copy any parts of it if you find this at all useful.

### Requirements Python:
1. Python 3.10+ must be installed
2. Pip install customtkinter (run in terminal)
3. python entry.py (run in terminal)

OR 

1. Download and run 'Wesnoth_Campaign_Creator'
